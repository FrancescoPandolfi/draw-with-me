# ✏️  Switcho - Collaborative drawing app

## Work in progress app: [Click me](https://sketcho.vercel.app/) 

Made with:

- React
- Redux
- Node.js
- Socket.io
- Canvas API


## Running this Repo

Download the source, open a terminal to the source folder and ...

```
npm install
npm start
```
Then navigate to "backend" folder and ...

```
npm install
npm start
```
